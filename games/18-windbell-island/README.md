# 风铃岛纪行

治愈系横版动作冒险 / 轻魂类。八座岛区、十二位守护灵、八封信、双结局。

## 特色

- 体力、风滚、轻/重击和「风铃共鸣」
- 露营茶桌：恢复、存档、升级、投递信件
- 失败会掉落风铃碎片；再次失败只损失上一份碎片的一半
- 生物和守护灵被安抚后平静离场，不使用血腥表现
- 商业手绘水彩背景、角色、十二守护灵和双结局插画
- 留岛 / 远航双结局，通关后解锁守护灵回忆挑战

## 操作

| 键 | 动作 |
|----|------|
| A / D | 移动 |
| J | 风铃轻击 |
| K | 纸伞重击 |
| Space | 风滚（13 帧无敌） |
| L | 风铃共鸣（12 帧窗口） |
| F | 喝热茶 |
| E | 互动 / 菜单确认 |
| W / S | 菜单选择 |
| Esc | 关闭露营菜单 |

## 路线

蒲公英港 → 雨铃森林 → 蜜橘坡 → 云上牧场 → 蓝玻潮池 → 风车谷 → 星眠山 → 晨曦灯塔
