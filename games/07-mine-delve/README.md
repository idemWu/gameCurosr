# 矿灯深途

分层挖矿 Roguelite：管理生命与灯油，完成十五项任务并抵达 B25 底层。

## 操作
- 方向键 / WASD：移动（消耗灯油）
- 蓝=铜矿 · 青=秘银 · 黄=金矿 · 红=怪物 · 绿=下一层
- 「升井」：返回上一层（B1 可查看营地结算）
- 「休息站补给」：第 1、5、10… 层可用金币换灯油
- 右下角 🔊：静音
- 「菜单」：继续 / 重开 / 清档

## 体验要点
- 灯油光照与 HP/油条 HUD
- 挖矿粒子、受击闪红、任务完成反馈
- 本地存档，刷新可续玩

## 时长
约 **18–28 分钟**

## 启动
```bash
python3 -m http.server 8080
# http://localhost:8080/games/07-mine-delve/
```
