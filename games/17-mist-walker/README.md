# 雾港行者（2D 动作魂类）

高难度横版魂类。**12 区互联 · 18 个多阶段 Boss（含 4 隐藏/中 Boss）· 3 结局 · NG+ · 无尽挑战塔**。

## 操作
| 键 | 动作 |
|----|------|
| A / D | 移动 |
| J / K | 轻击 / 重击（耗体力） |
| Space | 翻滚（10 帧无敌） |
| L | 弹反（基础 8 帧窗口） |
| F | 喝回复瓶 |
| Q | 投掷刀（商店购买） |
| E | 互动 / 篝火菜单确认 |
| W / S | 篝火菜单选择 |
| Esc | 关闭菜单 |

## 魂类规则
- 死亡掉魂原地，可拾回；再死消失
- 篝火：回满、存档、敌人重置、升级加点（生命/体力/力量循环）、购买技能与护符（贩售点：沉船腹地、盐渍矿道）
- 高难度：敌人 3–5 击致死、Boss 多阶段提速、无难度选项
- 通关最终 Boss「溺神·残躯」后三选一结局 → 进入 NG+（每周目敌人 +35%）
- 任一结局后，雾码头篝火可进入「挑战塔」无尽层数刷分

## 12 区路线
雾码头 → 沉船腹地 → 灯塔崖 → 旧城地窖(毒) → 盐渍矿道 → 钟楼回廊 → 溺水教区 → 雾墙荒原 → 黑潮堡垒 → 星骸观台 → 深渊祭坛(毒) → 溺神王座

## 时长口径
主线约 30 小时（普通玩家含死亡重试；熟练魂类玩家约 12–15 小时）。详见 `docs/playtime/17-mist-walker.md`。
