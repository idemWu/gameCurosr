# 雾港行者（2D 动作魂类）

高难度横版魂类。当前进度：**S0 战斗核心切片**（雾码头 + 教学 Boss「码头巡守」）。

## 操作
| 键 | 动作 |
|----|------|
| A / D | 移动 |
| J | 轻击（耗体力） |
| K | 重击 |
| Space | 翻滚（10 帧无敌） |
| L | 弹反（8 帧窗口，成功进入处决） |
| F | 喝回复瓶 |
| E | 互动：篝火休息存档 / 拾回掉落魂 / Boss 后过门 |

## 魂类规则
- 死亡：魂掉落原地，回收返还；再次死亡旧魂消失
- 篝火：回满血、补瓶、存档、重置小怪
- 敌人伤害高（3–5 击死）

## 路线体量（目标）
主线 12 区 / 18–20 Boss / 约 30 小时（普通玩家）；当前为第 1 区切片。
