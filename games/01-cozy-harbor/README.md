# 港湾日记

治愈向海边小镇日常。七天日程、三区域、八位镇民，完成待办迎来灯火庆典。**无种田。**

## 操作
- WASD / 方向键：移动（网格步进，带平滑插值）
- E：与附近 NPC 交谈并完成对应待办
- 右下角 🔊：静音开关
- 「菜单」：继续 / 重开 / 清档

## 体验要点
- 时段天空与海面氛围
- NPC 待办感叹号、对话气泡、好感浮动字与粒子
- 本地存档，刷新可续玩

## 时长
约 **12–25 分钟**（视探索节奏）

## 启动
```bash
python3 -m http.server 8080
# http://localhost:8080/games/01-cozy-harbor/
```
