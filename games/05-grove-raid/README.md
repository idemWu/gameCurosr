# 林间轻旅

俯视森林探索：打怪、开宝箱、推进十五件林务，穿越五个区域抵达星落营地。

## 操作
- WASD / 方向键：移动
- 靠近敌人：自动攻击（冷却环显示就绪）
- 靠近宝箱：自动开启
- 右下角 🔊：静音
- 「菜单」：继续 / 重开 / 清档

## 体验要点
- 森林背景、敌人与宝箱绘制
- 受击闪红、攻击冷却环、任务完成浮动字
- 本地存档，刷新可续玩

## 时长
约 **15–25 分钟**

## 启动
```bash
python3 -m http.server 8080
# http://localhost:8080/games/05-grove-raid/
```
