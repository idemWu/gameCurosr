# 雾港行者 · 时长预估（高难度口径）

普通玩家（含死亡重试与练怪）：

| 阶段 | 内容 | 约小时 |
|------|------|--------|
| 前段 | z01–z03 + Boss 1–4（学习期死亡多） | 5–7 |
| 中段 | z04–z06 + Boss 5–8（毒区/召唤/隐藏骑士） | 7–9 |
| 后段 | z07–z10 + Boss 9–14（连段/远程压力） | 8–10 |
| 终局 | z11–z12 + Boss 15–18 + 三阶段终 Boss | 6–8 |
| 升级/回收/探索 | 刷魂加点、找隐藏、护符技能 | 4–6 |
| **主线合计** | | **≈30（±20%）** |

熟练魂类玩家：约 12–15 小时。  
NG+ 每周目：8–15 小时；挑战塔：无上限。

> 高难度死亡重试是魂类体裁的核心体验，计入主线时长；数值曲线（Boss 血量/伤害递增、回复瓶稀缺）在 `content/` JSON 内可调。
