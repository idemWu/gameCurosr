# 游戏组合说明（Longplay）

本仓库目标：交付 **16 个浏览器小游戏**，可玩精致小品（多数约 **10–30 分钟**，带存档），覆盖四大类，**不做种菜/种田**。

## 体量定义（Longplay）

每款游戏必须：

1. 打开 `index.html` 即可玩（无需构建、无强依赖外网 CDN）
2. 有开始 / 继续（读档）/ 结局界面
3. 有清晰目标与结局；强调美术/音效/操作反馈（禁止纯空等灌水）
4. `localStorage` 存档（刷新可续玩）+ 可清档重开
5. 中文 UI 与 README（含阶段时长预估）
6. 目录自包含在 `games/<id>-<slug>/`（可复制 `shared/longplay`）

## 非目标

- Steam 级 3A / 联机 / 云存档 SDK
- 种田 / 作物生长玩法
- 抄袭商业游戏素材与商标

## 四大类内容量下限（Longplay）

| 类型 | 下限 |
|------|------|
| 治愈休闲 | 多日/多委托进度 + 图鉴或好感线，有完整小循环与结局 |
| 轻冒险 | 大地图或多层/40+ 关量级内容 + Boss/结局 |
| 卡牌/放置/策略 | 战役章节或多地图波次 + 成长树 |
| 益智/超休闲 | 40–60+ 关或等价长线订单/配方章节 |

## 技术栈

- Vanilla HTML + CSS + JavaScript
- 存档助手：`shared/longplay/save.js`
- 推荐内容驱动：`content/*.json`
