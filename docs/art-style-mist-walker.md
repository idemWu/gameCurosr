# 《雾港行者》程序化像素美术规范

## 方向

关键词：**海雾、冷铁、孤火、宗教废墟、低饱和、强轮廓**。

全部游戏内画面由 Canvas 2D 程序化绘制，不依赖外部图片。逻辑画布为
480×270，CSS 使用 `image-rendering: pixelated` 放大。

## 角色语言

- 行者：冷灰兜帽、青黑披风、单眼面罩、苍蓝长刃。
- 普通敌人：士兵 / 骑士 / 施法者 / 野兽 / 射手五类轮廓。
- 精英：金色细框，体型较宽。
- Boss：32×56 以上体型，冠角和超大武器；终 Boss 使用深海蓝和发光青。
- 攻击预警：出招前切换为亮红；硬直切换为金黄。

## 环境层次

1. 天空渐变和月轮。
2. 远景建筑/桅杆/塔楼剪影（慢速视差）。
3. 近景废墟和窗口微光（中速视差）。
4. 动态雾带。
5. 可行走平台、毒池、门和篝火。
6. 角色、弹道、命中粒子。

## UI

- 像素化长条血量、体力和 Boss 生命。
- 冷铁边框、内阴影、等宽字体。
- 菜单使用双层细框和金色菱形焦点。
- 不使用圆润手游卡片风。
