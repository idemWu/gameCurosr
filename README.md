# gameCurosr · 16 款长线休闲小游戏

浏览器小游戏作品集。覆盖四大类共 **16 款**，主线目标约 **30–60 分钟**（含本地存档），**不做种菜/种田**。

## 快速试玩

```bash
python3 -m http.server 8080
# 总览：http://localhost:8080/preview/
```
